Rules of the game :
1. Batting or bowling is decided by a toss which is generated randomly in python
2. The batsman has to put a target score for the bowler so that he/she can chase the score .
3. If batsman and bowler puts same number , then it means the bowler has taken the wicket of the batsman .
4. Then the chasing is done by the opponent 
5. Played human vs computer( Numbers , toss and decision randomly using random module in python )
